# il-linguaggio-c
Questa repository raccoglie le mie soluzioni personali agli esercizi di fine capitolo del libro Il linguaggio C di Paul e Harvey Deitel (nona edizione). L’obiettivo è documentare il mio percorso di studio del linguaggio C, condividere codice pulito e ben commentato, e creare un punto di riferimento utile per chi sta affrontando lo stesso testo.
